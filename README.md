# ユーモアAIチャット Kikkot (Hack-1_2025)

「同じ話を繰り返すと、人の代わりにAIがユーモアのある返答をしてくれる」音声対話型のデスクトップアプリケーションです。
ハッカソン「Hack-1_2025」にて開発し、部門賞を受賞しました。

## プロジェクトの概要
日常のコミュニケーションにおいて「何度も同じ話をしてしまう」という課題を、技術とユーモアの力でポジティブな体験に変えることを目指して制作しました。

## 主要な機能
1. **話者識別 (Speaker Identification)**
   音声特徴量（MFCC）を抽出し、K-Meansクラスタリングを用いることで、誰が話しているかをリアルタイムに判別します。話者ごとにUIの表示色を切り替えます。

2. **文脈検索 (Context Search)**
   過去の会話ログをTF-IDFとコサイン類似度で検索し、現在の発言が過去のどの話題と類似しているかを特定します。

3. **ユーモア返答生成 (Humor Generation)**
   Google Gemini APIを活用し、過去の文脈を考慮した返答を生成します。「同じ話をしている」と直接指摘するのではなく、文脈を汲み取ったユーモアある返答を行うよう制御しています。

## 技術スタック
- **言語**: Python 3.10+
- **GUI**: PyQt5
- **AI/ML**: Google Gemini API, OpenAI Whisper, Scikit-learn, librosa
- **ライブラリ**: SpeechRecognition, gTTS, NumPy
